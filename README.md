## Official Resources

- [Postman Learning Center](https://learning.postman.com) - Official documentation covering setup, scripting, and advanced features.
- [Postman API Network](https://www.postman.com/explore) - Public collections and API integrations.
- [Postman Blog](https://blog.postman.com) - Product updates and technical insights.
- [Postman GitHub](https://github.com/postmanlabs) - Official open-source tools and SDKs.

## Tools & CLI

- [Newman](https://github.com/postmanlabs/newman) - Official CLI runner for collections in CI/CD pipelines.
- [Postman CLI](https://learning.postman.com/docs/postman-cli/postman-cli-overview) - First-party CLI with authentication and performance testing.
- [postman-code-generators](https://github.com/postmanlabs/postman-code-generators) - Generate code snippets from requests.

## Collections & Converters

- [openapi-to-postman](https://github.com/postmanlabs/openapi-to-postman) - Convert OpenAPI specs to Postman Collections.
- [postman-collection](https://github.com/postmanlabs/postman-collection) - JavaScript SDK for working with collections programmatically.
- [All-Things-Postman](https://github.com/DannyDainton/All-Things-Postman) - Extensive examples and scripts.

## Tutorials & Learning

- [Postman Beginner's Course](https://www.youtube.com/watch?v=VywxIQ2ZXw4) - Free YouTube course on API testing fundamentals.
- [API Testing and Development with Postman, 2nd Edition](https://www.packtpub.com/en-us/product/api-testing-and-development-with-postman-9781804617908) - Comprehensive guide to API testing with Postman.
- [Postman Flows: Next Generation of API Development](https://blog.postman.com/postman-flows-the-next-generation-of-software-development) - Introduction to Flows for API automation.

## CI/CD Integration

- [GitHub Actions](https://learning.postman.com/docs/integrations/available-integrations/ci-integrations/github-actions) - Run collections in GitHub workflows.
- [postman-cli-action](https://github.com/postmanlabs/postman-cli-action) - Official GitHub Action for Postman CLI.
- [Jenkins Integration](https://learning.postman.com/docs/integrations/available-integrations/ci-integrations/jenkins) - Integrate Newman in Jenkins pipelines.
